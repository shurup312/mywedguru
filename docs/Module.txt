Модуль представляет собой логически сгруппированные файлы.
Сами модули располагаются в папке modules. Каждый модуль находится в папке с именем модуля, написанным с маленькой буквы.
Так же модуль обязательно должен быть прописан в настройках в
system\defines.php в разделе modules.
Для примера модуль пользователей с папкой /modules/users
Должен быть зарегистрирован как
...
"users"=>[

],
...

В папке модуля могут быть следующие папки
controllers
    содержит контроллер Controller.php, унаследованный от \system\core\Controller (более подробно в документации по контроллерам)
views
    файлы отображений
actions
    классы-экшены (подробней в документации Actions)
assets
    классы, описывающие ресурсы (подробнее читать Assets)
models
    модели, необходимые для работы модуля (читаем Model документацию)
modules
    вложенные модули.

     В случае. есои модуль вложен в другой модуль, то ничего по структуре не меняется, для вложенных модулей структура остается такой же.
Регистрируются вложенные модули в system\defines.php следующим образом
'{родительский модуль}'=>[
    'modules'=>[
        '{дочерний модуль}'=>[],
    ]
],

для примера
return [
    ...
	'modules' => [
	    ...
		'realty'=>[
			'modules'=>[
				'object'=>[],
				'clients'=>[],
			]
		],
        ...
