В системе есть два режима вывода ошибок:
- для разработчиков
- для продакшена

При выводе ошибок для разработчиков выводится описание ошибки, стэк вызова файлов и коды файлов с подсвеченной строкой.
Пример: http://f5.s.qip.ru/19gZJH3rx.png

В режиме продакшена выводится только текст ошибки и сообщение к пользователям.
Пример: http://f6.s.qip.ru/19gZJH3ry.png

За переключение между режимами отвечает константа __DEBUG_MODE__ в файле system\defines.php. Она может принимать значения true|false
соответственно включая и выключая расширенный вывод ошибок.
